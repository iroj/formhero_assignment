Instructions

To demonstrate your knowledge of JavaScript and AngularJS (or your ability to quickly learn how to write AngularJS Directives / Components) we have put together this mini application.

We would like you to create a directive/component that is a custom Date Input.

This directive/component should be called fh-date-input, should allow fh-data-input to be used as an attribute to instantiate the directive and should meet the following requirements:

  • req 1
  • req 2

The next tab, "Your Solution", is already set up to use the new directive, and contains a variety of uses of the directive that should demonstrate it's functionality.

To give you some additional guidance on what the user interface should look like, we are including the following images that visually describe what we expect the widget to look like in various states:

1.

When the input is first loaded, and there is no value set for it.
2.

When the user first click into the month field, a popup/modal should appear, allowing the selection of the month.
3.

When the month is selected, the view in the modal/popup should automatically switch to allow the selection of the day

The back-arrow should return the user to the selection for the month.
4.

Once the day is selected, the view should change to allow the user to pick the year. It should allow navigation between decades above the list of years.

The back-arrow should return the user to the selection for the day.
